# 🎓 Student Lifestyle Analysis

## 📌 Objective
This project analyzes student performance and explores how different lifestyle factors influence academic outcomes.

The goal is to understand whether behaviors such as alcohol consumption, social activity, study time, and attendance have measurable effects on student grades.

---

## 📊 Dataset
- Student Performance Dataset
- Includes features such as:
  - Absences
  - Study time
  - Alcohol consumption (Dalc, Walc)
  - Social activity (goout)
  - Academic performance (G3)

---

## 🔍 Analysis
The project focuses on exploring relationships between lifestyle variables and student performance:

- Absences vs Final Grade (G3)
- Alcohol Consumption vs Academic Performance
- Social Activity (Going Out) vs Grades
- Study Time vs Academic Outcomes
- Pass/Fail classification based on final grade

---

## 📈 Key Findings
- Students with higher absence rates tend to have lower academic performance  
- Increased alcohol consumption is associated with slightly lower grades  
- Frequent social activity may negatively impact performance, although moderate levels show limited effect  
- Study time has a positive but relatively weak correlation with grades  
- Academic performance is influenced by multiple lifestyle and behavioral factors, rather than a single variable
